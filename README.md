# マチマチマッチング

> 「やりたいこと」を双方向で表明し、自動でマッチングされる地域密着型活動マッチングサービス

## コンセプト

「街」で「待ち」でマッチング。

- **街（マチ）**: 地域密着型。東京・仙台など特定エリアでの活動マッチング
- **待ち（マチ）**: 「やりたいこと」を表明しておくだけでオファーが届く

## 特徴

- 🎯 **ゆるい募集**: 日時・場所が確定してなくても投稿OK
- 🔄 **双方向マッチング**: 募集者も参加希望者も意思表示できる
- 🤖 **自動提案**: 「やりたいこと」に基づいてユーザーを自動提案
- 👆 **ワンタップオファー**: 気軽に誘える・誘われる

## 技術スタック

- **Frontend**: Vue 3 + TypeScript + Pinia + Tailwind CSS
- **Backend**: Node.js + Fastify + tRPC
- **Database**: PostgreSQL + Prisma ORM
- **Realtime**: Socket.io
- **Auth**: Supabase Auth
