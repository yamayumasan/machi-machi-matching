# 機能一覧

## 2つの投稿タイプ

### タイプ1：募集（募集者向け）

```
【募集】来週末ボドゲやりませんか？

- カテゴリ：ボードゲーム
- 日時：来週末のどこか
- 場所：仙台駅周辺
- 人数：2〜5人
- ステータス：募集中

→ 参加者を集めて実行する意思がある
```

### タイプ2：やりたいこと表明（参加希望者向け）

```
【やりたい】ボドゲやりたい気分🎲

- カテゴリ：ボードゲーム
- 時期：今週〜来週
- エリア：仙台
- ステータス：誘ってほしい

→ 自分で募集を立てるほどではないが、誘われたら参加したい
```

### 両者の違い

| 項目 | 募集 | やりたいこと表明 |
|------|------|----------------|
| **目的** | 人を集めて実行する | 誘われたい・マッチしたい |
| **詳細度** | 日時・場所をある程度指定 | ざっくりでOK |
| **アクション** | オファーを送る側 | オファーを受ける側 |
| **有効期限** | 手動 or 自動締め切り | 自動で期限切れ（例：1週間） |
| **複数設定** | 1つずつ | 複数同時にOK |

## 「やりたいこと表明」詳細仕様

### 入力項目

| 項目 | 必須 | 例 |
|------|------|-----|
| カテゴリ | ★★★ | ボードゲーム |
| 時期 | ★★☆ | 今週末 / 来週 / いつでも |
| ひとこと | ★☆☆ | 「初心者だけど興味あり！」 |
| 有効期限 | 自動 | 1週間で自動消去（延長可） |

### 表示形式

```
┌─────────────────────────────────────┐
│ 🎲 ボドゲやりたい                    │
│                                     │
│ 📅 来週くらい                        │
│ 📍 仙台                              │
│ 💬 「カタンやってみたい！」          │
│                                     │
│ ⏰ あと5日で期限切れ  [延長] [削除]  │
└─────────────────────────────────────┘
```

### 複数同時設定

```
今やりたいこと：
  🎲 ボドゲ（来週まで）
  ☕ カフェ巡り（今週末）
  🏃 ランニング（いつでも）
```

## 募集の締め切り制御

| パターン | 動作 |
|---------|------|
| 手動締め切り | 募集者が任意のタイミングで締め切り |
| 自動締め切り | 最大人数に達したら自動で締め切り |
| 期限締め切り | 設定した日時で自動締め切り（Phase 2） |
